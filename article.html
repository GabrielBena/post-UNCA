A Minimalist Google Docs workflow for Distill.pub

The typical process of a Distill.pub article creation requires all contributors and editors to be familiar with version control (specifically Git) and web development, which sets an unnecessarily high barrier for entry to accept edits and suggestions from reviewers. This repository contains an article template and instructions for setting up a workflow based on Google Docs which exports the article html to GitHub. The workflow also uses a minimalistic Python web-server that automatically fetches new changes from GitHub and recompiles the article. We also provide a minimal Jekyll configuration that allows you to see changes live on GitHub Pages.

Fork and rename the https://github.com/znah/gdocs_distill_template repository. Duplicate this Google Document for your article. Update “github_repo” variable in above table to point to your GitHub repository. Click "HTML Export" -> "Run Export", provide a GitHub API key.
